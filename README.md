# 🗂️ Portofolio — Iffat Ayman Ahnaf

> Live site: **[iffat-ayman-ahnaf.github.io](https://iffat-ayman-ahnaf.github.io)**

Personal portfolio website built with pure HTML, CSS & JavaScript — no frameworks, no dependencies beyond Google Fonts and Font Awesome.

---

## ✨ Features

- **Bilingual (EN / ID)** — toggle between English and Indonesian, persisted via `localStorage`
- **Responsive** — mobile-first layout with dedicated desktop grid, bottom nav bar on mobile, fullscreen drawer
- **Smooth scroll reveals** — `IntersectionObserver` powered entrance animations
- **Tab-based project panel** — separate Data and Design project views
- **Contact form** — powered by [FormSubmit](https://formsubmit.co), zero backend required
- **Active section tracking** — top nav and bottom nav highlight as you scroll
